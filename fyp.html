FINAL YEAR PROJECT
SMART COURSE ORGANIZER

Smart Course Organizer for JTMK Department - An Web based system that developed to organize student management. In the existing system, After the exam results come out, all students need to print their exam result slip, after that the slip needs to be given to the academic advisor. Beside that, the Academic Advisor can see if there are any students who failed the semester and need to repeat the semester. In addition, the academic advisor had difficulties when want to check the student records or slip due stack files stored in the rack. This will cause problem in term of time because must check one by one files. However, by using this system student can manage their academic info by themselves and less dependency on Academic Advisor. In addiction, this system have List of Courses for JTMK Department every semester and also can calculate CGPA
